# Markdown ファイルベースの Kanban ビュー

## 概要

Markdown ファイルをタスク管理の単位として扱い、それを Kanban ボードとして可視化するビューを実装する。

## 動機

- シンプルでポータブルなタスク管理がしたい
- Markdown という普遍的なフォーマットでタスクを管理したい
- frontmatter によるメタデータでタスクの状態管理をしたい

## ユースケース

1. ユーザーはアプリ起動時に作業ディレクトリを選択する
2. 選択したディレクトリ内の `.md` ファイルを走査し、frontmatter の `status` に基づいて Kanban ボードにカードとして表示する
3. 各カードはタイトル（ファイル名）とボディの内容を表示する
4. カードを別のカラムにドラッグ & ドロップすると、frontmatter の `status` が更新される

## 非目標（今回のスコープ外）

- ファイルの新規作成機能
- マークダウンのリッチエディタ
- 複数ユーザー対応
- ステータス以外の frontmatter フィールドの編集 UI
